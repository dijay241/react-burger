## Установка проекта

Скачайте репозиторий:
### `git clone git@github.com:dijay241/react-burger.git`

Для отладки локально запустите:
### `yarn start`

Для запуска тестов хранилища запустите:
### `yarn test`

Для запуска e2e-тестов запустите:
### `yarn cypress:open`

Для деплоя проекта запустите:
### `yarn build`
